TRANSLITERATOR.exe fajl se nalazi u dist folderu. Njega treb pokrenuti.

Transliterator je program koji prevodi uneseni tekst na glagoljicu. Program nije zamisljen kao nesto sto prevodi velike
tekstove,nego neke manje inpute. Postoji i opcija da se prijevod spremi kao screenshot.
programski jezik: python 2.7, uz pomoc Tkinter modula. OOP pristup.

Program sam prvo napisao iz zabave, krajem 2014. godine, a nakon dorade je postao i mojim zavrsnim projektom za kolegij objektno i vizualno programiranje. (ffzg, odsjek za informacijske i komunikacijske znanosti)
